<> Grundlagen von HTML, CSS und JS <>

  • HTML

    HTML, oder Hypertext Markup Language, ist eine Markup-Sprache für das Internet, die die Struktur von Webseiten definiert. [Quelle]

  • CSS

    CSS steht für Cascading Style Sheets und ist die Sprache, mit der die visuelle Darstellung von Webseiten gestaltet wird. CSS ist die Sprache, die Webbrowsern mitteilt, wie die verschiedenen Teile einer Webseite dargestellt werden sollen[Quelle]

  • Javascript

    JavaScript ist eine leichtgewichtige Programmiersprache, die von Webentwicklern häufig verwendet wird, um dynamische Interaktionen zu Webseiten, Anwendungen, Servern und sogar Spielen hinzuzufügen. [Quelle]
